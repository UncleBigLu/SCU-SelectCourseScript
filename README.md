## 使用:

- 在电脑上安装`python3`运行环境

- 将运行目录切换至本脚本根目录

- 安装脚本所需库:
    你可以使用下面**两种方法**

    1. 输入`pip install -r requirements.txt`
    2. 手动安装以下库：
        - requests
        - BeautifulSoup
        - Pillow
        - **lxml**

- 运行`python selectCourse.py`,根据提示执行即可.
